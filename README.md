## :page_with_curl: Внешнее описание проекта

Проект позволяет настроить аватарку по клику на нее и профессию пользователя по клику на кнопку с карандашом. Вы можете добавить свои карточки через кнопку с плюсиком или посмотреть чужие. У вас есть возможность удалить свою карточку по нажатию на кнопку с корзиной. Существует возможность добавления лайков по клику на сердечко, еще вы видите количество лайков от других пользователей. На клик по картинке открывается ее полноразмерный вариант. Выйти из просмотра картинки или открытой формы можно по нажатию на крестик.

## :sparkles: Ключевые особенности проекта

Интегрировать проект [Место](https://github.com/MaximSinyukov/mesto) на `React`. Использовать функциональные компоненты. Подписать необходимые компоненты на `context`. Использовать `React Hooks`. Создать адаптивный сайт. Учесть наиболее распространенные разрешения экранов. Ограничить длину имени и профессии троеточием. Настроить: открытие/закрытие попапов через кнопку "крестика", формы, запросы к внешнему API. Реализовать постановку и снятие лайка. Настроить удаление собственных карточек.

## :wrench: О работе с проектом

1. Перед работой установите необходимые зависимости через - `npm install`
2. Для локального запуска проекта используйте - `npm run start`
3. Для итоговой сборки проекта воспользуйтесь - `npm run build`
4. Для деплоя сайта на gh-pages воспользуйтесь - `npm run deploy`
